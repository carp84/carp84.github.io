Apache ecosystem

Open infrastructure at production scale.

My open-source work centers on data infrastructure that has to survive real production scale.

Long-Running Threads

My Apache work has grown from HBase and Hadoop-era storage infrastructure into Flink state backends, lakehouse formats, remote shuffle, execution acceleration, and AI-era data systems. The common thread is building open infrastructure that is technically deep, community-governed, and proven under demanding production workloads.
