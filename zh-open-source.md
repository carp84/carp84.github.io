## 长期主线

我的 Apache 工作从 HBase、Hadoop 时代的大数据存储基础设施，逐步延展到 Flink 状态后端、湖格式、远程 Shuffle、执行加速以及 AI 时代的数据系统。贯穿其中的主线，是把技术深度、社区治理和真实生产负载下的可靠性结合起来。

## Apache 角色

ASF Member
: 2021 年成为 ASF Member，持续参与 Apache 社区与 ALC Beijing 相关活动。

PMC Member
: Apache Flink、HBase、Paimon、Celeborn、Gluten、Fluss；HBase Committer 与 PMC 工作始于 2016 年。

Champion
: Apache Paimon、Apache Celeborn、Apache Fluss 孵化阶段 Champion。

Mentor
: Apache Amoro、Apache GraphAr Mentor。
