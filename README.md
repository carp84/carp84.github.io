# Yu Li Personal Homepage

This is a lightweight static GitHub Pages site for Yu Li / 李钰.

## Structure

English pages:

- `index.html`
- `open-source.html`
- `products.html`
- `speaking.html`

Chinese pages:

- `zh.html`
- `zh-open-source.html`
- `zh-products.html`
- `zh-speaking.html`

Shared styling lives in `styles.css`. The profile image uses the public GitHub avatar URL for `carp84`.

## Publish on GitHub Pages

1. Create a GitHub repository named `carp84.github.io`.
2. Push these files to the repository's default branch.
3. Visit `https://carp84.github.io`.

If using a different repository name, enable Pages in the repository settings and choose the default branch as the source.
